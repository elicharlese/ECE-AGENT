## üöÄ Installation from Source

### Prerequisites
- Node.js 18+ and npm
- Git

### Build Instructions
```bash
# Clone the repository
git clone https://github.com/elicharlese/AGENT.git
cd AGENT

# Install dependencies
npm install

# Start development server
npm run dev

# Build desktop app (requires Electron)
npm run desktop:build
```
